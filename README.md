# TEP-LENS: Strong Lensing Time Delay Closure Tests

This repository contains the analysis pipeline and manuscript for **Paper 14** of the Temporal Equivalence Principle (TEP) series.

## Overview
This paper tests the TEP framework using a "route-closure" or "triangle test" on multiply imaged transients (such as the lensed supernova SN H0pe) and strongly lensed quasars (from the TDCOSMO sample). 

Under General Relativity (GR), light traveling along different paths through a gravitational potential arrives with relative time delays determined strictly by the geometric path difference and the Shapiro time delay. Under TEP, because the effective speed of temporal propagation is modified in deep gravitational potentials, the delays should exhibit a scale-dependent closure residual that scales with the lens mass and the specific photon paths.

## Repository Structure
- `data/`: Raw light curves, lens models, and compiled catalogs.
- `scripts/steps/`: Sequential analysis pipeline.
- `scripts/simulations/`: Theoretical TEP lensing predictors.
- `scripts/figures/`: Publication-quality plotting scripts.
- `results/`: Pipeline outputs and figures.
- `site/`: Source code for the interactive HTML manuscript.
- `14manuscript-tep-lens.md`: The compiled Markdown manuscript.
