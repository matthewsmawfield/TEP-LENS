OGLE-II Photometry of QSO 2237+0305 (1997-2000)

Files in this directory:

README             - this file
fchart.fts         - finding chart (FITS format)
fchart.ps.gz       - finding chart (gzipped Postscript)
lcurve_flux.ps.gz  - light curve of images A-D in flux units (gzipped PS)
lcurve_mag.ps.gz   - light curve of images A-D in maginitude units (gzipped PS)
phot.dat           - photometry data in text format; columns contain:
                     col. 1 - date of observation (YY-MM-DD.ddd)
                     col. 2 - Heliocentric Julian Date - 2450000
                     col. 3 - airmass
                     col. 4-11 - V magnitudes and errors of images A,B,C,D
