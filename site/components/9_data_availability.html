Data Availability & Reproducibility

Open Science Statement

This work follows strict open-science practices. All results are fully reproducible from raw data using the documented pipeline. All numerical results, figures, and statistics are generated by deterministic Python scripts processing real observational data.

Repository & Code

GitHub Repository: github.com/matthewsmawfield/TEP-LENS

The repository contains a deterministic, version-controlled analysis pipeline with 26 analysis steps for geometric route-closure tests in multiply-imaged supernovae.

Reproduction Instructions

Quick Start (Full Reproduction)

# 1. Clone repository
git clone https://github.com/matthewsmawfield/TEP-LENS.git
cd TEP-LENS

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run complete pipeline (20+ steps)
python scripts/steps/run_all_steps.py

# 4. Build manuscript
cd site
npm install
npm run build

Detailed Pipeline Steps

The analysis pipeline consists of 26 deterministic steps. Each step produces JSON outputs and logs for full traceability:

Data Acquisition & Closure Tests (Steps 01-04)
  • step_01_fetch_snh0pe_data.py — Fetch SN H0pe light curve data from public release; parses multiply-imaged supernova magnitudes and time delays
  • step_02_gr_closure.py — GR geometric route-closure test; computes residuals for GR baseline with SN Refsdal and SN H0pe
  • step_03_tep_closure.py — TEP closure analysis; computes TEP-predicted route closure and compares to observations
  • step_04_plot_closure.py — Generate closure residual plots (Figure 4 baseline vs residual)
TDCOSMO & H0 Analysis (Steps 05-10)
  • step_05_tdcosmo_shear.py — TDCOSMO external convergence shear analysis; tests environmental systematics
  • step_06_alpha_sensitivity.py — α₀ parameter sensitivity analysis; tests how closure residuals depend on TEP coupling
  • step_07_observed_vs_predicted.py — Observed vs predicted time-delay comparison; quantifies TEP predictive accuracy
  • step_08_new_evidence.py — Compile new evidence from independent lens systems
  • step_09_precision_roadmap.py — Precision roadmap analysis; projects future constraints from JWST lensed SNe
  • step_10_h0_tension.py — H0 tension analysis in lensing context; compares TEP predictions to local vs CMB H0
Model Dependence & Robustness (Steps 11-15)
  • step_11_model_dependence.py — Model dependence analysis; tests sensitivity to lens model assumptions
  • step_12_microlensing_robustness.py — Microlensing robustness tests; quantifies microlensing contamination effects
  • step_13_bayes_model_comparison.py — Bayesian model comparison; computes Bayes factors for GR vs TEP lens models
  • step_14_external_chain_ingestion.py — External chain ingestion; imports TDCOSMO/H0LiCOW MCMC chains for joint analysis
  • step_15_informed_inflation.py — Informed inflation analysis; tests TEP effects on time-delay cosmography precision
External Data & Completeness (Steps 16-20)
  • step_16_independence_tier_significance.py — Independence tier significance; statistical tests for multiple independent lens systems
  • step_17_directional_odds.py — Directional odds analysis; tests TEP predictions for specific image parity configurations
  • step_18_external_dataset_registry.py — External dataset registry; catalogs all public lensed SN light curves
  • step_19_tdcosmo2025_ingestion.py — TDCOSMO-2025 chain ingestion; imports latest time-delay cosmography constraints
  • step_20_external_completeness_synthesis.py — External completeness synthesis; combines all lensing constraints with TEP
Latest Robustness Upgrades (Steps 21-26)
  • step_21_gr_fom.py — GR figure-of-merit analysis; quantifies GR vs TEP model performance
  • step_22_gamma_closure.py — Gamma closure test; extended geometric closure with TEP screening corrections
  • step_23_gr_full.py — Full GR baseline analysis; comprehensive GR-only model comparison
  • step_24_tep_closure_validation.py — TEP closure validation; cross-checks with independent TEP implementations
  • step_25_snh0pe_reanalysis.py — SN H0pe reanalysis with updated TEP predictions
  • step_26_final_synthesis.py — Final synthesis and manuscript consistency check

Each step produces JSON outputs with full metadata in results/outputs/, and execution logs are written to logs/ with timestamps for complete traceability. Run all steps via: python scripts/steps/run_all_steps.py
